jsonp 가 뭡니까?
	==> 원래 Ajax통신은
			동일한 서버에 있는 프로그래만 요청할 수 있어요!
			(Ajax통신을 요청하는 자바스크립가 있는 파일과 
					요청하는 서버의 프로그램이 같은 프로젝트에 있어야해요)
			--> 동일한 도메인
			
	==> 그런데 만약에 외부에 있는(즉, 다른 서버에 있는)
			데이터를 Ajax통신을 하려면
			jsonp 방식으로 해야 합니다.
			
-------------------------------------------------------------------


Ajax통신으로 요청되는 서버가 
응답하는 데이터의 형식은 3가지

==> 
csv		: 데이터를 콤마로 구분하여 응답하는 방식입니다.
xml		: 그데이터 무엇인지 마크하여 응답하는 방식입니다.
json	: 데이터를 자바스크립트 객체의 형태로 응답하는 방식입니다.
