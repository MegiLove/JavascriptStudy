Ajax통신으로 요청되는 
	서버가 응답하는 형식
		csv		: 값을 컴마로 분리하여 응답하는 방식
		xml		: 그 값이 무슨값인지 mark하여 응답하는 방식
		json	: 값을 자바스크립트 객체의 형태로 응답하는 방식		
		
		
-----------------------------------------------------------------------
xstream
	자바객체를 xml 문자열로 만들어주는 라이브러리 입니다.
	
	
gson
	자바객체를 json 문자열로 만들어주는 라이브러리 입니다.
